# Simple-Name-Username-Age-Database
A simple database that stores name/Surname and age.
It has 4 functions.
1. It can add a new person to the database.
2. It can write an orginized list with all people in the database
3. It can serch for a person, with only name/Surname/age or a combination. And sends out a list of all the peolple with that name/surname/age.
4. It can remove a person from the database

This is written only in c# and was served to be a preperation to working with SQL.
